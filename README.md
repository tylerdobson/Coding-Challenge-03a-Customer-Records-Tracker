# Customer Records Tracker

  **Course:** ISM 3232 — Business Application Development  **Assignment:** Coding Challenge 03a
  **Author:** Tyler Dobson

  ## Overview

  A small JavaScript program that manages customer records for a small business
  using **arrays** and **objects**. Each customer is stored as an object
  containing a name, email, and a list of purchases, and the full customer list
  is managed as an array.

  ## What It Demonstrates

  - Building compound data structures (an array of objects)
  - Adding and removing records with `.push()` and `.shift()`
  - Updating object properties and nested array data
  - Iterating with `.forEach()` and formatting output with template literals

  ## How to Run

  Requires [Node.js](https://nodejs.org/).

  ```bash
  node cc_3.js
